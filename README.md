## 🍽️ Catering Management System

A complete **web-based Catering Management System** built using **PHP, MySQL, HTML, CSS, and JavaScript**. This system helps manage catering services efficiently by providing separate dashboards for **Admin, Staff, and Customers**.

---

 Features

Admin Panel

 Manage users (customers & staff)
 View and control bookings/orders
 Manage menu and catering packages
 Generate reports
 Dashboard overview

Staff Panel

 View assigned orders
 Update order status
 Manage daily tasks

 Customer Panel

 User authentication (Login/Signup)
 Browse menu & packages
 Book catering services
 View order history



 Tech Stack

Frontend HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Server: XAMPP / Apache

---

## 📂 Project Structure

* `admin/` → Admin dashboard & controls
* `customer/` → Customer interface
* `staff/` → Staff dashboard
* `auth/` → Authentication system
* `config/` → Database configuration
* `database/` → SQL file
* `includes/` → Reusable components (header, footer, sidebar)
* `assets/` → CSS & JS files

## ⚙️ Installation Guide

1. Clone the repository

```bash
git clone https://github.com/your-username/catering-system.git
```

2. Move project to XAMPP

```
C:\xampp\htdocs\
```

3. Import database

* Open phpMyAdmin
* Create database
* Import `database/catering.sql`

4. Configure database

* Open `config/db.php`
* Update credentials

5. Run project

```
http://localhost/catering-system
```
